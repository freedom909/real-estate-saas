# AI Platform Context

## Architecture
- **Entry**: `src/ai-platform/index.ts` — Express + Apollo Server (subgraph) on port 4200
- **DI**: tsyringe container, registered in `src/ai-platform/container/registers/ai-platform.register.ts`
- **Auth**: `getUserFromToken(req)` extracts JWT from `Authorization: Bearer <token>` header
- **Flow**: Resolver → AIRequestFactory → AIPlatformOrchestrator → AgentFactory → DomainAgent

## Critical DI Fix
- `BookingRegister()` registers `IBookingRepository` → `SequelizeBookingRepository` under `Symbol.for("BookingRepository")`
- Without it, `BookingAgent` → `CreateBookingUseCase` → `@inject(TOKENS_BOOKING.repository.bookingRepository)` resolved to `undefined`
- tsyringe internally accessed `.constructor` on `undefined` → crash
